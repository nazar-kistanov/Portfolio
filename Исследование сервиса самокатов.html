Исследование сервиса по аренде самоатов GoFast¶


Вы аналитик популярного сервиса аренды самокатов GoFast. Вам передали данные о некоторых пользователях из нескольких городов, а также об их поездках и подписках. Проанализируйте данные и проверьте некоторые гипотезы, которые могут помочь бизнесу вырасти.

Цель исследования:

  1. Опишите и визуализируйте общую информацию о пользователях и поездках:
  • частота встречаемости городов;
  • соотношение пользователей с подпиской и без подписки;
  • возраст пользователей;
  • расстояние, которое пользователь преодолел за одну поездку;
  • родолжительность поездок.
  1. Осуществить подсчёт выручки
  • найдите суммарное расстояние, количество поездок и суммарное время для каждого пользователя за каждый месяц.
  • посчитать помесячную выручку, которую принёс каждый пользователь.
  1. Проверить гипотезы
  • тратят ли пользователи с подпиской больше времени на поездки?
  • проверить превышают ли расстояние оптимальное с точки зрения износа самоката
  • будет ли помесячная выручка от пользователей с подпиской по месяцам выше, чем выручка от пользователей без подписки.
  • из за обновления серверов, снизилось количество обращений в техподдержку.
    Ход исследования
    Данные о пользователях получу из /datasets/users_go.csv, данные о поездках получу из /datasets/rides_go.csv и данные о подписках получу из /datasets/subscriptions_go.csv. О качестве данных ничего неизвестно поэтому сделаю обзор и предобработку данных.


Таким образом мое исследование пройдет в 7 этапов:

  • Загрузка данных и обзор данных
  • Пердобработка данныхз
  • Исследовательский анализ данных
  • Объединение данных
  • Подсчет выручки
  • проверка гипотез
  • общий вывод </div>

Шаг: Проверка гипотез¶

Важно понять, тратят ли пользователи с подпиской больше времени на поездки? Если да, то пользователи с подпиской могут быть «выгоднее» для компании. Проверьте гипотезу. Используйте исходные данные о продолжительности каждой сессии — отдельно для подписчиков и тех, у кого нет подписки.¶

  • H0 - Время по подписке = время без подписки
  • H1 - Время по подписке > время без подписки

Вывод: По подписку ultra ежемесячная выручка с пользователей выше чем по подписке free

Представьте такую ситуацию: техническая команда сервиса обновила сервера, с которыми взаимодействует мобильное приложение. Она надеется, что из-за этого количество обращений в техподдержку значимо снизилось. Некоторый файл содержит для каждого пользователя данные о количестве обращений до обновления и после него. Какой тест вам понадобился бы для проверки этой гипотезы?¶

Ответ: Нам понеадобится тест "Гипотеза о равенстве средних для зависимых (парных) выборок", где

  • H0: μ1 = μ2
  • H1: μ1 != μ2
  • где μ1 и μ2 - средние значения количество обращений в техподдержку до и после изменения. Нам понадобился бы метод scipy.stats.ttest_rel()
